# Arquitetura de Computadores (MIPS Assembly)

[cite_start]Este repositório contém a resolução de exercícios da disciplina de Arquitetura de Computadores na **PUC-SP**

## Exercícios Desenvolvidos
1. **Soma de dígitos do RA**: Realiza a soma individual de cada algarismo do RA dos integrantes.
2. **Comparação de valores**: Identifica o maior e o menor número entre dois registradores ($t0 e $t1).
3. **Condicional If/Else**: Tradução de lógica em C para Assembly, testando igualdade entre variáveis.
4. **Laço de Repetição (While)**: Implementação de um loop que realiza somas sucessivas e incrementa um contador.

## Como Executar
1. Faça o download do simulador **MARS** (MIPS Assembler and Runtime Simulator).
2. Abra o arquivo `.asm` desejado no simulador.
3. Clique em **Assemble** (ícone da ferramenta).
4. Execute o código e acompanhe os valores nos registradores na aba lateral direita.
